# Code Reading Instructions

When reading or searching code in this project, always use jcodemunch tools instead of reading files directly:

- Use `index_folder` at the start of a session to index the project
- Use `get_repo_outline` or `get_file_outline` to explore structure
- Use `search_symbols` and `get_symbol` to look up specific classes/methods
- Use `get_context_bundle` to fetch a symbol plus its imports
- Use `search_text` for full-text search across files

Do NOT use the Read tool to read source files when a jcodemunch tool can retrieve the needed information instead.
